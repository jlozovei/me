Flavored Markdown

Editor de markdown para o browser, com sabores.

Um editor de markdown para o navegador, Flavored Markdown mostra como os seus arquivos .md vão ser renderizados em diferentes plafatormas: como o GitHub, GitLab, Bitbucket, Azure e várias outras.

Feito utilizando Vue e vuex, a ideia do site veio enquanto eu comparava o estilo de visualização de arquivos markdown entre GitHub e Bitbucket. Como um bom escritor de readmes, eu sempre gostei de saber como esses arquivos seriam renderizados, e essa ferramenta me ajudou a visualizar isso.

Esse projeto é open source e está no GitHub. O projeto ainda usa o GitHub Actions como CI e o GitHub Pages como host; o domínio customizado é fornecido pelo Google Domains.
