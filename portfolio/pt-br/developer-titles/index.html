Developer Titles

Títulos incríveis de desenvolvedor para descrições, biografias e status.

Após ver vários títulos, no mínimo, hiperbólicos no LinkedIn, fui inspirado pelo Pseudo Design Titles, tive a ideia de replicar os pseudo títulos para o mundo de tecnologia e desenvolvimento.

A ideia do site é brincar com a forma que os desenvolvedores definem seus cargos ou posições.

Esse projeto é open source e está no GitHub. O projeto ainda usa o GitHub Actions como CI e o GitHub Pages como host; o domínio customizado é fornecido pelo Google Domains.

  • Ano: 2019
  • Stack: htmlcssjswebpack
